## Codebase Patterns

### Findings validator heading contract
- **Where**: `scripts/validate-findings.ts:33-39`, `context/critics/*/findings.md`
- **Pattern**: Critic artifacts must use an exact `## Issues` heading; `## Issues identified` fails `bun run validate:findings`.
- **Example**: `## Issues`

### Memory substrate module shape
- **Where**: `scripts/memory.ts:1-20`
- **Pattern**: Keep the memory substrate in a standalone `scripts/` module with a first-line orchestrator-only guard comment and named exports only.
- **Example**: `export type MemoryFilter = Partial<Pick<MemoryEvent, "week" | "actor" | "event">>;`
